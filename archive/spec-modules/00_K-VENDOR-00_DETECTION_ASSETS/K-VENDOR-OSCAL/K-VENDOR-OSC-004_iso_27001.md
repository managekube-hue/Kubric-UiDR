# K-VENDOR-OSC-004 -- ISO 27001:2022 OSCAL Mapping

## Overview

ISO/IEC 27001:2022 is the international standard for information security management systems (ISMS). Kubric maps ISO 27001 Annex A controls to detections and agent capabilities to support continuous compliance for enterprise and international tenants.

## Annex A Control Mapping

| Control | Title | Kubric Evidence Source |
|---|---|---|
| **A.5.1** | Policies for information security | KAI-Risk policy assessment reports |
| **A.5.23** | Information security for cloud services | Nuclei cloud templates (NUC-003) |
| **A.8.1** | User endpoint devices | CoreSec agent telemetry, Watchdog health |
| **A.8.5** | Secure authentication | K-SVC JWT/RBAC, Authentik MFA |
| **A.8.7** | Protection against malware | YARA rules (YAR-002), Suricata ET |
| **A.8.8** | Management of technical vulnerabilities | VDR EPSS scoring, Nuclei CVE scans |
| **A.8.15** | Logging | CoreSec OCSF event stream |
| **A.8.16** | Monitoring activities | NetGuard IDS, KAI alert pipeline |
| **A.8.20** | Network security | NetGuard flow analysis, Suricata rules |
| **A.8.24** | Use of cryptography | NetGuard TLS version monitoring |
| **A.8.28** | Secure coding | Nuclei API templates (NUC-004), CAPEC mapping |

## Data Structure

ISO 27001 controls are expressed as an OSCAL catalog with cross-references to NIST 800-53:

```json
{
  "catalog": {
    "metadata": { "title": "ISO/IEC 27001:2022 Annex A" },
    "groups": [
      {
        "id": "a8",
        "title": "Technological Controls",
        "controls": [
          {
            "id": "a-8-7",
            "title": "Protection against malware",
            "props": [{ "name": "nist-mapping", "value": "SI-3" }]
          }
        ]
      }
    ]
  }
}
```

## Kubric Integration

K-SVC exposes `/api/v1/compliance/iso-27001` for ISO-specific posture reports. The NOC dashboard renders a control heatmap showing implementation status across all 93 Annex A controls. Gap reports feed into KAI-Risk for residual risk calculation per the ISMS scope.

ISO 27001 compliance status is included in tenant billing reports generated by the K-SVC Stripe integration for MSSP customers requiring ISO certification support.
